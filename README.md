# Erth: Smart Digital Legacy System

Erth is a smart digital legacy system designed to help users securely manage and transition their digital assets and legacies. Built with a modern tech stack, the system integrates advanced processing to ensure security, compliance, and seamless user experience.

## Features
* **Smart Legacy Management:** Securely store and automate the transition of digital assets.
* **OCR & Document Processing:** Advanced Optical Character Recognition for verifying identity and documents.
* **NLP Integration:** Natural Language Processing capabilities for smarter data handling and user interactions.

## Tech Stack & Technologies
* **Frontend:** Flutter (Dart)
* **Backend & Database:** PostgreSQL
* **AI & Analytics:** NLP (Natural Language Processing) & OCR (Optical Character Recognition)

## Compliance & Standards
The system is built with a strong focus on cybersecurity frameworks and digital regulations, aligning with:
* National Cybersecurity Authority (NCA) standards.
* Saudi Data and AI Authority (SDAIA) regulations.
* Personal Data Protection Law (PDPL) in Saudi Arabia.
